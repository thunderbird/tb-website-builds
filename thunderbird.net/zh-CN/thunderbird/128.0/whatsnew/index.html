欢迎来到 Supernova 的进化

通过革命性的代码改进Nebula 为更快地发布新功能、增强稳定性、改善生活质量以及改进视觉效果打下了基础。

快速而流畅

感受智慧代码带来的不同消息列表交互更快、email space 更灵敏、Thunderbird 更稳定可靠。

卡片视图

新版设计让浏览邮件更快、布局更美观、话题体验也得到改进。另外邮件卡片的高度会根据您的设置自动调整从而使一切看起来都恰到好处。

强调色

增强主题兼容性尤其适用于 Ubuntu 和 Mint 上的 Linux 用户。 Thunderbird 将与您的桌面环境更好地融合与系统的主题色无缝衔接。

文件夹窗格

一系列改进包括更快地呈现和搜索统一文件夹、选择多个文件夹的功能以及更好地召回消息状态。

Rust 革命

我们解决了将 Rust 引入 Thunderbird 的主要障碍。这为更好的代码、桌面与移动设备间的功能共享以及更快的开发铺平了道路。对于我们的开发者和用户来说这是一个转折点。

简化的菜单导航具有更好的视觉提示和更少的认知负担让您的电子邮件体验更高效、更愉快。

Windows 原生通知

已修复且功能完整点击通知将其解除将 Thunderbird 带到前台并选择消息。通知现在会在 Thunderbird 关闭时消失。

上下文菜单改进

界面重新组织以获得更顺畅的体验主要操作现在显示为图标便于快速访问。

自定义账户颜色

应大众的要求您现在可以自定义账户图标的颜色。在撰写邮件时这些颜色也会出现在“发件人”选项中。

“Nebula”带来前所未有的闪耀邮件体验。在接下来的几个月里期待更多的更新和实用的新功能出现。
