欢迎见证 Supernova 的进化

Nebula 大刀阔斧改进代码，为加快提供新功能、提高稳定性、改善使用体验、改良视觉效果打下了坚实基础。

毫不拖泥带水

敬请感受精妙代码造就的美妙变化！消息列表反应更迅捷，邮件空间也更灵敏，整个 Thunderbird 都更稳定可靠。

卡片视图

新版设计为浏览邮件大提速，也让界面布局更美观、话题功能更好用。此外，邮件卡片高度会根据您的设置自动调整，每一处都恰到好处。

强调色

主题的兼容性获得增强，使用 Ubuntu 和 Mint 的 Linux 用户对此应该尤为受用。Thunderbird 与您的桌面环境将更加契合，能够完美对接系统的强调色。

文件夹窗格

一众改进现已就位：综合文件夹的加载和搜索快上加快，邮件话题状态的检索得到优化，还可以同时选中多个文件夹了。

掀起 Rust 革命

我们扫清将 Rust 引入 Thunderbird 的主要障碍，为今后提高代码质量、同步桌面端与移动端功能、节省开发时间开了好局。无论对于开发者还是用户来说，这都是一次深刻变革。

菜单导航经过简化，视觉提示更为合理，操作逻辑更合直觉。让收发邮件再高效一点，再愉快一些。

Windows 原生通知

相关问题得到解决，功能已完全恢复！点击通知即可将其清除、切换 Thunderbird 至前台显示，并选择相应消息。关闭 Thunderbird 时，通知现也会随之消失。

上下文菜单改进

上下文菜单经过重新排列，用起来更加顺手。常用操作现以图标形式展现，方便快速取用。

自定义账户颜色

顺应广大用户呼声，账户图标的颜色现可自选了。撰写邮件时，相关颜色也会显示于”发件人“部分。

“Nebula”所带来的邮件体验如此夺目，前所未见。未来几个月，一大波更新及实用新功能将陆续登场，敬请期待。
