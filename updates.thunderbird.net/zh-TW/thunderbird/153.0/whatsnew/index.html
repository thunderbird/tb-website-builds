Thunderbird 153

歡迎使用 Meadow。這個版本讓 Thunderbird 提供 Exchange 的原生支援、全新的「帳號中心」以及各種自訂選項，讓 Thunderbird 變得更強大、更個人化，也讓 Thunderbird 能有您的風格。

更簡單就能註冊帳號

全新的「帳號中心」，讓設定帳號更快、更直覺，還可以自動連結行事曆與通訊錄。

建立通訊錄

Meadow 會掃描您現有的帳號，並在 Thunderbird 設定您的本機及/或遠端通訊錄。

Microsoft Exchange 支援

現在起，使用者不需安裝任何附加元件，即可以原生方式設定 Microsoft Exchange 帳號，讀寫或管理郵件。

強調色彩

您也可以在「外觀」標籤使用新的強調色彩設定來自定您的色彩。

自訂您的通知

直接在作業系統的原生通知當中將郵件設為已讀、刪除、標示為垃圾信等動作。

還有更多功能…

用瀏覽器進行 OAuth 驗證

現在起，Thunderbird 支援直接透過您的預設瀏覽器進行 OAuth 驗證。

使用 Thundermail 登入

若您擁有 Thundermail 帳號，可以從「帳號中心」點擊，直接登入 Thunderbird。

改進的信件匣排序功能

改進過的介面與更好的視覺標示，讓排序郵件比以往更簡單。

Bug 修正與改進

有數以千計的 Bug 修正與效能改進，讓您有更滑順、可靠的 Thunderbird 使用體驗。

升級到新的 Thunderbird 正式發行版本

不用等一年就能使用下個版本 — 每月都有穩定、可靠的新版本推出。
