Thunderbird d aseɣẓan ilelli yeldin, mahlent yifḍan n imţekkiyen deg umaḍal. Atan wayen issefk ad teẓreḍ:

Tanfiwin n telɣut web n Thunderbird

Thunderbird iseqdac tanfiwin n telɣut Web ("Services") akken ad k-d-imudd kra n temhalin di lqem-agi imsin n Thunderbird akken d-ittwaskan daw-a. Ma tebɣiḍ ur teseqdaceḍ ara yiwet si tanfiwin-agi neɣ ma tebɣiḍ ur tqebleḍ ara tiwtilin-agi, tzemreḍ ad tekkseḍ armad n temhalt neɣ tanfa.
