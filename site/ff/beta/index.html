Know the Risks First

Thunderbird Beta is for testing.

Thunderbird Beta is not intended for general use. It can be unstable and in rare cases result in data loss. We strongly recommend you read the knowledge base article before proceeding.

Be Prepared

  • Regularly backup your beta profile to prevent data loss.
  • Anticipate frequent updates and ensure proper installation into a beta-specific directory.
  • If using mail filters, run them in only one profile.
  • Be aware that some add-ons may not be compatible with beta.
  • Avoid using POP accounts with beta. If necessary, configure to leave messages on the server indefinitely.
