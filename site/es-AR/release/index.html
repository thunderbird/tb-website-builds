Bienvenido a Thunderbird

¿Por qué necesitamos donaciones?

Thunderbird ya no está financiado por Mozilla. Afortunadamente existe una comunidad activa que lo mantiene y desarrolla constantemente. Para que el proyecto sobreviva mucho tiempo, se requiere financiamiento.

El financiamiento de Thunderbird se usa para mantener la infraestructura y el personal para llevar la aplicación a nuestras decenas de millones de usuarios. Para que Thunderbird sobreviva y evolucione, necesitamos su apoyo y le pedimos una donación. Todo el dinero donado irá directamente a costear el desarrollo y la infraestructura de Thunderbird.

Done a Thunderbird

Thunderbird es un destacado cliente de correo electrónico y calendario para todas las plataformas de código abierto, gratuito, tanto para el uso empresarial como el personal. Quremos mantenerlo seguro y que sea aun mejor. Una donación nos permitirá contratar desarrolladores, pagar por la infraestructura y continuar mejorando.

¡Thunderbird está financiado por usuarios como vos! Si te gusta Thunderbird, por favor considerá hacer una donación La mejor manera de asegurar su supervivencia es donando.
